## AWS Shield – CLF-C02 Scope 🛡️🌐

**AWS Shield** is a managed service that protects applications against **DDoS (Distributed Denial of Service) attacks**.

It is provided by  
Amazon Web Services.

---

## What Is a DDoS Attack?

A DDoS attack tries to:

- Flood a system with massive traffic
    
- Make the application unavailable
    

AWS Shield helps detect and mitigate these attacks automatically.

---

## Types of AWS Shield (Exam Important)

### 1️⃣ Shield Standard (Free)

- Automatically enabled for all AWS customers
    
- Protects against common network and transport layer attacks
    
- No extra cost
    

### 2️⃣ Shield Advanced (Paid)

- Enhanced DDoS protection
    
- 24/7 access to AWS DDoS Response Team (DRT)
    
- Cost protection during scaling caused by attacks
    

---

## Works With

- Amazon CloudFront
    
- Elastic Load Balancing
    
- Route 53
    
- EC2
    

---

## Key Exam Points 🎯

- Protects against DDoS attacks
    
- Shield Standard is automatic and free
    
- Shield Advanced is paid
    
- Improves availability and security
