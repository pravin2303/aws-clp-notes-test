# AWS Snowball

## 📌 Category
Data Migration / Data Transfer Service

---

## 🚚 What is AWS Snowball?

AWS Snowball is a physical data transfer device used to move large amounts of data into or out of AWS.

It is part of the AWS Snow Family.

---

## 🎯 Why Use Snowball?

- Transfer hundreds of TB or petabytes of data
- Limited internet bandwidth
- Large-scale data center migration
- Faster than internet for huge datasets

---

## ⚙️ How It Works

1. AWS ships the Snowball device to you  
2. You copy data to the device  
3. Ship the device back to AWS  
4. AWS uploads data into Amazon S3  

---

## 🧠 Use Cases

- Data center migration  
- Disaster recovery bulk transfer  
- Large database migration  
- Media archive transfer  

---

## 📊 Snowball vs DataSync

| Feature | Snowball | DataSync |
|----------|-----------|-----------|
| Transfer Type | Physical Device | Online Transfer |
| Internet Required | No | Yes |
| Best For | Very Large Datasets | Continuous Sync |

---

## 📦 Snowball vs Snowmobile

- Snowball → TB to PB scale  
- Snowmobile → Exabyte-scale migration
