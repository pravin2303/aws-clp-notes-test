## 🚀 AWS CodeDeploy – CLF-C02 Scope

**AWS CodeDeploy** is a service used to **automatically deploy applications** to AWS compute services.

It helps developers update applications **quickly and reliably**.

---

# 🎯 What CodeDeploy Does

✔ Automates application deployments  
✔ Reduces downtime during updates  
✔ Supports rolling deployments  
✔ Integrates with other AWS DevOps tools

---

# Where It Can Deploy Applications

CodeDeploy can deploy applications to:

- Amazon EC2 instances
    
- AWS Lambda
    
- On-premises servers
    

---

# Example

A company wants to update a web application running on multiple EC2 instances.

Instead of updating each server manually, they use **CodeDeploy** to deploy the update automatically.

---

# CodeDeploy vs Elastic Beanstalk

|Feature|CodeDeploy|Elastic Beanstalk|
|---|---|---|
|Purpose|Application deployment automation|Application platform management|
|Control|More control over deployment|Fully managed environment|
|Focus|DevOps deployment|Application hosting|

👉 “Automate application deployment” → CodeDeploy  
👉 “Deploy and manage web apps quickly” → Elastic Beanstalk

---

# 🎯 Exam Keywords

- **Automated deployment** → CodeDeploy
    
- **Deploy application updates** → CodeDeploy
    
- **DevOps deployment tool** → CodeDeploy
