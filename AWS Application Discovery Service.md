## 🔎➡️☁️ AWS Application Discovery Service – CLF-C02 Scope

**AWS Application Discovery Service** helps you **collect information about on-premises data centers** before migrating to AWS.

It is used in the **planning phase of migration**.

---

# 🎯 What It Does

✔ Discover servers and applications  
✔ Collect configuration details  
✔ Measure performance data  
✔ Identify dependencies between applications

---

# Why It Is Important

Before migrating to AWS, companies need to understand:

- What servers are running
    
- How applications depend on each other
    
- Resource usage patterns
    

👉 “Discover and assess before migration” → Application Discovery Service

---

# Example (Exam Scenario)

If question says:

“Company wants to gather information about its on-premises servers before migrating to AWS.”

Correct answer → Application Discovery Service.

---

# Discovery Service vs Migration Services

|Service|Purpose|
|---|---|
|Application Discovery Service|Assess & discover infrastructure|
|DMS|Migrate databases|
|Application Migration Service|Migrate servers|
|Snowball|Transfer large data physically|

---

# 🎯 Exam Keywords

- “Migration planning” → Application Discovery Service
    
- “Discover on-prem infrastructure” → Application Discovery Service
    
- “Collect server configuration data” → Application Discovery Service
