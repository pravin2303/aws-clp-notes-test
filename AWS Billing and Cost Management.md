## 💰 AWS Billing and Cost Management – CLF-C02 Scope

**AWS Billing and Cost Management** is the central service used to:

- View AWS bills
    
- Monitor usage
    
- Manage payments
    
- Control spending
    

It is where you manage all **account-level billing information**.

---

# 🎯 What It Provides

✔ View current & past invoices  
✔ Track service usage  
✔ Access payment history  
✔ Manage payment methods  
✔ Download billing reports

---

# Includes Important Tools

Inside Billing & Cost Management, you can access:

- AWS Cost Explorer → Analyze spending
    
- AWS Budgets → Set alerts
    
- AWS Cost and Usage Report → Detailed billing data
    

---

# Example (Exam Scenario)

If question says:

“Company wants to download monthly AWS invoice.”

Correct answer → AWS Billing and Cost Management.

If question says:

“Company wants alert when cost exceeds threshold.”

Correct answer → AWS Budgets.

If question says:

“Company wants to analyze past 6 months spending.”

Correct answer → Cost Explorer.

---

# Billing vs Support Concierge (Exam Concept)

|Feature|Billing & Cost Management|Support Concierge|
|---|---|---|
|View invoice|Yes|No|
|Payment management|Yes|No|
|Billing assistance|No|Yes|
