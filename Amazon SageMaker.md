## 🤖 Amazon SageMaker – CLF-C02 Scope

**Amazon SageMaker** is a service used to **build, train, and deploy machine learning (ML) models** on AWS.

It provides tools and infrastructure to make machine learning development easier.

---

# 🎯 What SageMaker Does

✔ Build machine learning models  
✔ Train models using large datasets  
✔ Deploy models for predictions  
✔ Manage the entire ML lifecycle

---

# Example

A company wants to predict **customer buying behavior** using machine learning.

They can use **SageMaker** to:

- Train the model
    
- Analyze data
    
- Deploy the model for predictions
    

---

# SageMaker vs Rekognition

|Service|Purpose|
|---|---|
|SageMaker|Build and train ML models|
|Rekognition|Analyze images and videos|

👉 “Create custom ML model” → SageMaker  
👉 “Detect faces in images” → Rekognition
