## 🚀 AWS CodeStar – CLF-C02 Scope

Amazon CodeStar is a service that helps developers quickly build, develop, and deploy applications on AWS.

It provides a single dashboard to manage the entire DevOps development workflow.

# 🎯 What CodeStar Does

✔ Quickly set up development projects
✔ Integrates multiple DevOps tools
✔ Provides project templates
✔ Tracks development progress

Integrated AWS DevOps Services

**CodeStar integrates services like:**

AWS CodeCommit → Store source code

AWS CodeBuild → Build applications

AWS CodeDeploy → Deploy applications

AWS CodePipeline → CI/CD automation

*Example*

A development team wants to start a new cloud application project quickly.

Using CodeStar, they can create a project template that automatically sets up:

Code repository

CI/CD pipeline

Deployment tools

CodeStar vs CodePipeline
Service	Purpose
CodeStar	Manage and set up DevOps projects
CodePipeline	Automate CI/CD workflow

👉 “Create and manage development project” → CodeStar
👉 “Automate build/test/deploy pipeline” → CodePipeline
