| Term              | Meaning                                                             | Cloud vs Traditional                   |
| ----------------- | ------------------------------------------------------------------- | -------------------------------------- |
| **Fixed-term**    | Payment for a **long contract period** (e.g., 1–3 years commitment) | Traditional IT / Reserved commitments  |
| **Pay-as-you-go** | Pay only for the **resources you actually use**                     | Core **cloud computing pricing model** |
| **Colocation**    | Hosting your own servers in a **third-party data center**           | Traditional infrastructure model       |
| **Planned**       | Capacity must be **planned in advance** before purchasing hardware  | Traditional IT approach                |
| **Variable cost** | Costs change depending on **usage level**                           | Cloud computing model                  |

## Key Differences

| Category       | Traditional IT               | Cloud Computing   |
| -------------- | ---------------------------- | ----------------- |
| Cost model     | Fixed-term                   | Pay-as-you-go     |
| Cost type      | Fixed cost                   | Variable cost     |
| Infrastructure | Colocation / own data center | Cloud provider    |
| Capacity       | Planned in advance           | On-demand scaling |
|                |                              |                   |
|                |                              |                   |
