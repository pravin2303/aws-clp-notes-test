## 🏗️ AWS Cloud Adoption Framework (CAF) – CLF-C02 Scope

**AWS Cloud Adoption Framework (CAF)** helps organizations **plan and successfully migrate to the AWS cloud**.

It provides **guidelines and best practices for cloud adoption**.

---

# 🎯 Purpose of AWS CAF

✔ Help companies move from on-premises to cloud  
✔ Align business and IT teams  
✔ Identify gaps in cloud readiness  
✔ Provide a structured migration strategy

---

# 6 Perspectives of AWS CAF

|Perspective|Focus|
|---|---|
|**Business**|Cloud strategy and value|
|**People**|Skills and organizational change|
|**Governance**|Compliance and policies|
|**Platform**|Cloud infrastructure and architecture|
|**Security**|Data and system protection|
|**Operations**|Manage and operate cloud workloads|

---

# Example

A company wants to migrate its data center to AWS but doesn't know where to start.

They use **AWS Cloud Adoption Framework** to:

- Assess readiness
    
- Plan migration
    
- Align teams and processes
    

---

# AWS CAF vs Well-Architected Framework

|Framework|Purpose|
|---|---|
|AWS CAF|Plan cloud adoption and migration|
|AWS Well-Architected Framework|Design and optimize cloud architecture|
