## ⚙️ OpsWorks – CLF-C02 Scope 🛠️

## AWS OpsWorks

**AWS OpsWorks** is a configuration management service that helps you **automate application deployment and server management**.

It uses tools like:

- Chef
    
- Puppet
    

---

## 🎯 What OpsWorks Does

✔ Automate server configuration  
✔ Manage infrastructure as code  
✔ Deploy applications  
✔ Manage application lifecycle

---

## When to Use (Exam Level)

- Need configuration management
    
- Want to use Chef or Puppet
    
- Automate server setup
    

---

## OpsWorks vs Systems Manager (Exam Concept)

|Feature|OpsWorks|Systems Manager|
|---|---|---|
|Uses Chef/Puppet|Yes|No|
|Patch EC2|Limited|Yes|
|Remote session|No|Yes|

👉 “Chef or Puppet” → OpsWorks  
👉 “Patch EC2 / Remote access” → Systems Manager
