## 📊 Amazon CloudWatch – CLF-C02 Scope

**Amazon CloudWatch** is a monitoring and observability service that helps you monitor **AWS resources and applications** in real time.

---

# 🎯 What CloudWatch Monitors

✔ CPU usage (EC2)  
✔ Memory & network metrics  
✔ Application logs  
✔ Custom metrics

---

# 🔔 CloudWatch Alarms (Very Important)

You can create **alarms** to:

- Send notifications
    
- Trigger Auto Scaling
    
- Take automated actions
    

👉 “Alert when CPU > 80%” → CloudWatch Alarm

---

# 🧠 What CloudWatch Helps With

- Performance monitoring
    
- Troubleshooting
    
- Automatic scaling
    
- Operational visibility
    

---

# CloudWatch vs CloudTrail (Exam Favorite)

|Service|Purpose|
|---|---|
|CloudWatch|Monitor performance & metrics|
|CloudTrail|Track API activity (who did what)|

👉 “High CPU usage alert” → CloudWatch  
👉 “Who deleted S3 bucket?” → CloudTrail

---
