## 🩺 AWS Health API – CLF-C02 Scope

**AWS Health API** provides **programmatic access to AWS Health information**.

It allows applications to **monitor AWS service health events automatically**.

---

# 🎯 What AWS Health API Does

✔ Retrieve AWS service health events  
✔ Detect service disruptions  
✔ Monitor scheduled maintenance  
✔ Integrate health alerts into monitoring systems

---

# How It Works

Instead of checking the dashboard manually, applications can use the API to:

- Query health events
    
- Automate alerts
    
- Monitor affected resources
    

It is the programmatic version of the  
**AWS Personal Health Dashboard**.

---

# Example

A company runs critical workloads on AWS.

They use the AWS Health API to:

- Automatically detect service outages
    
- Trigger alerts when resources are affected
    

---

# AWS Health API vs Service Health Dashboard

|Service|Purpose|
|---|---|
|AWS Health API|Programmatic access to health events|
|Service Health Dashboard|Public view of service status|

---

# 🎯 Exam Keywords

- **Programmatic health monitoring** → AWS Health API
    
- **Access service health events via API** → AWS Health API
    
- **Automated health alerts** → AWS Health API
