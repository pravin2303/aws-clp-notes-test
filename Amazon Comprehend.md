## 🧠 Amazon Comprehend – CLF-C02 Scope

**Amazon Comprehend** is an AWS service that uses **Natural Language Processing (NLP)** to analyze text and extract insights.

It helps understand **meaning and sentiment from text data**.

---

# 🎯 What Amazon Comprehend Can Do

✔ Detect sentiment (positive / negative / neutral)  
✔ Identify key phrases  
✔ Detect entities (names, places, organizations)  
✔ Language detection  
✔ Topic analysis

---

# Example

A company analyzes **customer reviews** to understand feedback.

Using **Amazon Comprehend**, they can:

- Detect customer sentiment
    
- Identify common issues
    
- Analyze product feedback
    

---

# Comprehend vs Translate vs Textract

|Service|Purpose|
|---|---|
|Amazon Comprehend|Analyze text meaning|
|Amazon Translate|Translate languages|
|Amazon Textract|Extract text from documents|

👉 “Sentiment analysis” → Comprehend  
👉 “Language translation” → Translate  
👉 “Extract text from scanned documents” → Textract
