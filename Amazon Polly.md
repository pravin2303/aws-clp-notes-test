## 🔊 Amazon Polly – CLF-C02 Scope

**Amazon Polly** is a service that converts **text into natural-sounding speech** using artificial intelligence.

It allows applications to **talk to users**.

---

# 🎯 What Amazon Polly Does

✔ Converts text to speech  
✔ Supports many languages  
✔ Provides realistic human-like voices  
✔ Used in voice-enabled applications

---

# Example

A mobile application reads **news articles aloud** to users.

The app sends text to **Amazon Polly**, and Polly converts it into spoken audio.

---

# Polly vs Lex vs Comprehend

|Service|Purpose|
|---|---|
|Amazon Polly|Text → Speech|
|Amazon Lex|Chatbots & voice interfaces|
|Amazon Comprehend|Text analysis|

👉 “Convert text to voice” → Polly  
👉 “Create chatbot” → Lex  
👉 “Analyze text sentiment” → Comprehend
