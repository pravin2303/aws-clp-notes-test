## 🧮 AWS Pricing Calculator

AWS provides an online tool called:

👉 **AWS Pricing Calculator**

It helps you:

- Estimate monthly AWS costs
    
- Calculate EC2, S3, RDS pricing
    
- Compare pricing options
    
- Plan budget before deployment
    

---

## 🔹 What You Can Calculate

You can estimate costs for:

- EC2 instances
    
- S3 storage
    
- RDS databases
    
- Lambda usage
    
- Data transfer
    
- And many other services
    

---

## 🔹 Example (CLF-C02 Type Scenario)

If question says:

“Company wants to estimate AWS cost before migrating workload.”

Correct answer → **AWS Pricing Calculator**

---

## 🔹 Pricing Calculator vs Cost Explorer

| Tool               | Purpose                    |
| ------------------ | -------------------------- |
| Pricing Calculator | Estimate future costs      |
| Cost Explorer      | Analyze past spending      |
| AWS Budgets        | Set alerts for cost limits |
