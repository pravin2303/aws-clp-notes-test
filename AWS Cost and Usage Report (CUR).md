## AWS Cost and Usage Report (CUR) – CLF-C02 Scope 📊💰

**AWS Cost and Usage Report (CUR)** provides the **most detailed billing data** for your AWS account.

It contains:

- Line-item usage details
    
- Resource-level cost data
    
- Pricing information
    
- Tags
    
- Discounts
    

---

## What CUR Is Used For

✔ Deep cost analysis  
✔ Custom billing reports  
✔ Integrate with analytics tools  
✔ Financial reporting

It is more detailed than Cost Explorer.

---

## How It Works (Simple Concept)

1️⃣ You enable Cost and Usage Report  
2️⃣ AWS generates detailed billing files  
3️⃣ Files are delivered to **Amazon S3**  
4️⃣ You analyze using:

- Athena
    
- Excel
    
- BI tools
    

---

## CUR vs Cost Explorer (Exam Favorite)

|Feature|Cost Explorer|Cost & Usage Report|
|---|---|---|
|Purpose|Visual cost analysis|Detailed raw billing data|
|Interface|AWS Console|Delivered to S3|
|Detail Level|Summary|Very detailed|

---

## Example (Exam Scenario)

If question says:

“Company needs detailed resource-level billing data for financial reporting.”

Correct answer → AWS Cost and Usage Report.

If question says:

“Company wants to view spending trends in console.”

Correct answer → Cost Explorer.
