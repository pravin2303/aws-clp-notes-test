## ⚙️ AWS Managed Services (AMS) – CLF-C02 Scope

**AWS Managed Services (AMS)** helps enterprises **operate their AWS infrastructure** following AWS best practices.

It is designed for companies that want AWS to help manage:

- Operations
    
- Security
    
- Monitoring
    
- Compliance
    

---

# 🎯 What AWS Managed Services Does

✔ Infrastructure operations management  
✔ Patch management  
✔ Backup management  
✔ Monitoring & incident response  
✔ Security controls

It follows the AWS Well-Architected Framework.

---

# Who Uses AMS?

- Large enterprises
    
- Organizations with strict compliance needs
    
- Companies lacking in-house cloud expertise
    

---

# Important Clarification (Exam Level)

AMS is NOT the same as:

- Managed AWS services like RDS or Lambda
    
- AWS Support Plans
    

It is a **separate enterprise service** that helps manage your AWS environment.

---

# AMS vs Support Plans

|Feature|AWS Managed Services|AWS Support Plans|
|---|---|---|
|Operates infrastructure|Yes|No|
|Troubleshooting help|Yes|Yes|
|Dedicated enterprise operations|Yes|Only Enterprise plan (TAM)|
|Best for large enterprises|Yes|Yes|

---

# Example (Exam Scenario)

If question says:

“Enterprise wants AWS to manage daily cloud operations and security.”

Correct answer → AWS Managed Services.

If question says:

“Company needs 24/7 technical support.”

Correct answer → Business or Enterprise Support Plan.

---

# 🎯 Exam Keywords

- “Enterprise cloud operations management” → AWS Managed Services
    
- “Operate AWS environment” → AMS
    
- “Follow best practices automatically” → AMS
