## 🖥️➡️☁️ AWS Server Migration Service (SMS) – CLF-C02 Scope

**AWS Server Migration Service (SMS)** was used to migrate on-premises servers to AWS.

⚠️ Important for exam awareness:  
SMS has been largely replaced by **AWS Application Migration Service (MGN)**.

---

# 🎯 What Server Migration Service Did

✔ Migrate virtual machines (VMs)  
✔ Automate server replication  
✔ Reduce manual migration effort

It helped move entire servers (not just databases).

---

# Modern Replacement

## AWS Application Migration Service (MGN)

Now recommended for:

- Lift-and-shift migration
    
- Moving applications to EC2
    
- Minimal downtime server migration
    

---

# SMS vs DMS (Exam Concept)

|Feature|Server Migration|Database Migration|
|---|---|---|
|Migrates|Entire servers|Databases only|
|Service|SMS / MGN|DMS|
|Use Case|Lift-and-shift|DB migration|

👉 “Migrate virtual machines to AWS” → Server Migration / MGN  
👉 “Migrate database with minimal downtime” → DMS

---

# 🎯 Exam Keywords

- “Lift-and-shift server migration” → Application Migration Service
    
- “Migrate virtual machines” → Server migration
    
- “Move on-prem server to EC2” → Application Migration Service
    

---

## One-Line Memory

> Server Migration = Move entire servers to AWS  
> DMS = Move databases only
