## Amazon Redshift – CLF-C02 Scope 📊🏢

**Amazon Redshift** is a **fully managed data warehouse service** used for **large-scale data analytics**.

It is designed for:

- Business intelligence (BI)
    
- Reporting
    
- Data analysis
    

---

## What Redshift Does

It allows you to:

- Analyze large datasets
    
- Run complex SQL queries
    
- Process structured and semi-structured data
    

---

## Key Characteristics (Exam Focus)

- Managed service
    
- Designed for analytics, not daily transactions
    
- Uses SQL
    
- Scales to petabytes of data
    

---

## When to Use Redshift

✔ Data warehousing  
✔ Business reporting  
✔ Analytics dashboards  
✔ Large historical datasets

---

## Redshift vs RDS (Exam Favorite)

|Feature|Redshift|RDS|
|---|---|---|
|Purpose|Analytics|Transactional database|
|Data Size|Very large (TB–PB)|Smaller workloads|
|Query Type|Complex analytical queries|Regular application queries|

---

## Example

Company collects sales data for 5 years.

They use Redshift to:

- Analyze trends
    
- Generate reports
    
- Create dashboards
