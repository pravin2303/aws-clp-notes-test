## Amazon DynamoDB – CLF-C02 Scope ⚡🗃️

**Amazon DynamoDB** is a **fully managed NoSQL database service** in AWS.

It is designed for:

- High performance
    
- Massive scalability
    
- Low latency
    

---

## What Type of Database?

- NoSQL (non-relational)
    
- Key-value and document database
    
- Serverless
    

Unlike RDS, it does NOT use tables with rows and columns like traditional SQL databases.

---

## Key Characteristics (Exam Focus)

- Fully managed
    
- Automatically scales
    
- High availability (multi-AZ by default)
    
- Millisecond latency
    

---

## When to Use DynamoDB

✔ Real-time applications  
✔ Gaming apps  
✔ IoT data  
✔ Mobile apps  
✔ Large-scale workloads

---

## DynamoDB vs RDS (Exam Favorite)

|Feature|DynamoDB|RDS|
|---|---|---|
|Type|NoSQL|Relational (SQL)|
|Schema|Flexible|Structured|
|Scaling|Automatic|Manual / configurable|
|Performance|Very high|High|

---

DynamoDB handles millions of requests per second with low latency.
