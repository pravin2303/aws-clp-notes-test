## Amazon ElastiCache – CLF-C02 Scope ⚡🗄️

**Amazon ElastiCache** is a **fully managed in-memory caching service**.

It is used to **improve application performance** by storing frequently accessed data in memory.

---

## Why Use ElastiCache?

Accessing a database can be slow.

Instead:

- Store frequently used data in cache
    
- Retrieve data much faster
    

Result → Lower latency + Better performance 🚀

---

## Supported Engines (High-Level)

- Redis
    
- Memcached
    

(No deep technical details required for CLF-C02.)

---

## Key Characteristics (Exam Focus)

- In-memory storage
    
- Very fast performance
    
- Reduces load on databases
    
- Improves application speed
    

---

## Example

E-commerce website:

- Product details stored in database
    
- Frequently viewed products stored in ElastiCache
    
- Users get faster response
    

---

## ElastiCache vs RDS (Exam Favorite)

|Feature|ElastiCache|RDS|
|---|---|---|
|Storage Type|In-memory|Disk-based|
|Purpose|Caching|Primary database|
|Speed|Very fast|Slower than cache|
