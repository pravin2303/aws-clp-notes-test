## 💻 AWS Command Line Interface (AWS CLI) – CLF-C02 Scope

**AWS CLI** is a tool that allows you to **manage AWS services using commands in a terminal or command prompt**.

Instead of using the AWS Management Console (web interface), you can control AWS resources from the command line.

---

# 🎯 What AWS CLI Does

✔ Manage AWS services from the terminal  
✔ Automate tasks using scripts  
✔ Create and manage AWS resources  
✔ Interact with AWS APIs

---

# Example Command

Example: List S3 buckets

aws s3 ls

Example: Launch an EC2 instance

aws ec2 run-instances

---

# How AWS CLI Works

1️⃣ Install AWS CLI on your computer  
2️⃣ Configure credentials (Access key + Secret key)  
3️⃣ Run commands to manage AWS resources

---

# AWS CLI vs Management Console

|Feature|AWS CLI|AWS Management Console|
|---|---|---|
|Interface|Command line|Web interface|
|Automation|Yes|Limited|
|Ease of use|Requires commands|User-friendly GUI|

👉 “Manage AWS from terminal” → AWS CLI  
👉 “Manage AWS from web browser” → Console
