## 🔍 Amazon Macie – CLF-C02 Scope

Amazon Macie is a data security and privacy service that uses machine learning to discover and protect sensitive data stored in Amazon S3.

**🎯 What Amazon Macie Does**

✔ Automatically scans S3 buckets
✔ Detects sensitive data
✔ Helps protect confidential information
✔ Alerts when sensitive data is exposed

*Types of Sensitive Data Detected*

Macie can detect:

Personal data (PII)

Credit card numbers

Passport numbers

Financial records

*Example*

A company stores files in
Amazon S3.

They want to know if any file contains personal or confidential data.

They use Amazon Macie to automatically scan and detect it.

Macie vs GuardDuty (Exam Concept)
Service	Purpose
Amazon Macie	Discover sensitive data in S3
Amazon GuardDuty	Detect security threats

👉 “Sensitive data discovery” → Macie
👉 “Threat detection” → GuardDuty

**🎯 Exam Keywords**

Sensitive data detection → Macie

Data privacy protection → Macie

Scan S3 for confidential information → Macie

One-Line Memory

Amazon Macie = Detect sensitive data in Amazon S3.
