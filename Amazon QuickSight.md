## 📊 Amazon QuickSight – CLF-C02 Scope

**Amazon QuickSight** is a **cloud-based Business Intelligence (BI) service** used to create:

- Dashboards
    
- Visualizations
    
- Reports
    
- Data insights
    

---

# 🎯 What QuickSight Does

✔ Connects to data sources  
✔ Creates interactive dashboards  
✔ Visualizes large datasets  
✔ Shares insights with users

---

# Common Data Sources

- Amazon S3
    
- Amazon Redshift
    
- RDS
    
- Athena
    
- External data sources
    

---

# Example (Exam Scenario)

If question says:

“Company wants to create interactive dashboards to visualize sales data.”

Correct answer → Amazon QuickSight.

---

# QuickSight vs Athena (Exam Concept)

|Service|Purpose|
|---|---|
|Athena|Query data in S3 using SQL|
|QuickSight|Visualize and create dashboards|

👉 “Run SQL query on S3 data” → Athena  
👉 “Create dashboard from data” → QuickSight
