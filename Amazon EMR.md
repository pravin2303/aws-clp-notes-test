## 🧠 Amazon EMR – CLF-C02 Scope

**Amazon EMR (Elastic MapReduce)** is a service used to **process large amounts of data (big data)** using distributed frameworks.

It runs big data tools like:

- Apache Hadoop
    
- Apache Spark
    
- Apache Hive
    
- Apache HBase
    

---

# 🎯 What Amazon EMR Does

✔ Processes big data  
✔ Performs large-scale data analysis  
✔ Runs distributed data processing frameworks  
✔ Scales automatically for big workloads

---

# How It Works (Simple Concept)

1️⃣ Data is stored in  
Amazon S3

2️⃣ EMR cluster processes the data

3️⃣ Results are generated for analytics

---

# Example

A company wants to analyze **petabytes of log data**.

They use **EMR with Apache Spark** to process the data.

---

# EMR vs Athena (Exam Concept)

|Service|Purpose|
|---|---|
|Athena|Run SQL queries on S3|
|EMR|Big data processing using Hadoop/Spark|

👉 “Run SQL on S3 logs” → Athena  
👉 “Process huge datasets with Spark/Hadoop” → EMR
