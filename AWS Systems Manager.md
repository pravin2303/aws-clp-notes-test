## AWS Systems Manager – CLF-C02 Quick Revision 🛠️📊

**AWS Systems Manager** helps you **manage, monitor, and operate your AWS resources at scale**.

It mainly manages:

- Amazon EC2 instances
    
- On-premises servers
    
- Hybrid environments
    

---

## 🎯 What It Does (Exam Focus)

✔ Remote instance management  
✔ Automate operational tasks  
✔ Apply OS patches  
✔ Store configuration parameters securely

---

## 🔹 Important Features (High-Level Only)

### 1️⃣ Session Manager

- Securely connect to EC2
    
- No need for SSH keys
    
- No need to open port 22
    

👉 “Secure remote access without SSH” → Session Manager

---

### 2️⃣ Patch Manager

- Automates OS patching
    

👉 “Automatically patch EC2 instances” → Systems Manager

---

### 3️⃣ Parameter Store

- Store configuration data
    
- Store secrets securely
    

---

## 🔥 Why It Is Important

- Improves security
    
- Reduces manual work
    
- Centralized control
    
- Works in hybrid environments
    

---

## 🧠 Systems Manager vs CloudWatch

|Service|Purpose|
|---|---|
|CloudWatch|Monitor metrics|
|Systems Manager|Manage & operate instances|

---

## 🎯 Exam Keywords

- “Remote instance management” → Systems Manager
    
- “Patch EC2 automatically” → Systems Manager
    
- “Securely connect without SSH” → Session Manager
    

---
