## 📜 AWS Artifact – CLF-C02 Scope

**AWS Artifact** is a service that provides **access to AWS security and compliance reports**.

Organizations use it to verify that AWS meets **regulatory and compliance standards**.

---

# 🎯 What AWS Artifact Provides

✔ Compliance reports  
✔ Security certifications  
✔ Agreements and audit documents

These documents help companies prove that AWS infrastructure follows security standards.

---

# Examples of Reports

AWS Artifact provides reports like:

- **SOC (Service Organization Control) reports**
    
- **ISO certifications**
    
- **PCI DSS compliance reports**
    

---

# Example Scenario

A company must prove to auditors that their cloud provider meets security standards.

They download compliance reports from **AWS Artifact**.

---

# AWS Artifact vs AWS Config

|Service|Purpose|
|---|---|
|AWS Artifact|Access compliance reports|
|AWS Config|Monitor resource configuration|
