## 🤝 AWS Partner Network (APN) – CLF-C02 Scope

The **AWS Partner Network (APN)** is a global community of companies that **partner with AWS** to provide solutions and services.

---

# 🎯 What APN Includes

APN partners help customers:

✔ Migrate to AWS  
✔ Build applications on AWS  
✔ Provide consulting services  
✔ Deliver managed services  
✔ Offer software solutions

---

# 🔹 Types of APN Partners (Exam Level)

### 1️⃣ Consulting Partners

- Help customers design and implement AWS solutions
    
- Example: Cloud migration consultants
    

### 2️⃣ Technology Partners

- Provide software solutions that run on AWS
    
- Example: Security tools, analytics software
    

---

# Why APN Is Important

If a company:

- Lacks AWS expertise
    
- Needs migration assistance
    
- Wants certified solution providers
    

They can work with APN partners.

---

# Example (Exam Scenario)

If question says:

“Company needs expert help to migrate workloads to AWS.”

Correct answer → AWS Partner Network partner.

---
