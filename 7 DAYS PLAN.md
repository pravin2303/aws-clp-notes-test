## **DAY 1 – Cloud Fundamentals & AWS Basics**

**Objective:** Build conceptual alignment. No shortcuts here.

### Study (2–3 hrs)

- What is Cloud Computing
    
- Benefits of Cloud Computing
    
- Cloud Service Models (IaaS, PaaS, SaaS)
    
- Deployment Models (Public, Private, Hybrid)
    
- Intro to **Amazon Web Services**
    

### Key Outcomes

- Understand _why_ companies move to cloud
    
- Know when to use IaaS vs PaaS vs SaaS
    

### Hands-On (Optional – 30 min)

- Create AWS Free Tier account
    
- Explore AWS Management Console
    

---

## **DAY 2 – AWS Global Infrastructure**

**Objective:** Understand how AWS delivers availability and low latency.

### Study (2 hrs)

- Regions
    
- Availability Zones
    
- Edge Locations
    
- Regional vs Global Services
    

### Must-Know Exam Logic

- High availability = **Multi-AZ**
    
- Low latency = **Edge locations**
    

### Hands-On (30 min)

- Open EC2 dashboard
    
- Switch between regions and observe services
    

---

## **DAY 3 – Compute Services**

**Objective:** Recognize compute options and use cases.

### Study (2–3 hrs)

- Amazon EC2
    
- EC2 instance types (conceptual)
    
- Amazon Lambda
    
- Elastic Beanstalk
    

### Exam Focus

- EC2 = full control
    
- Lambda = serverless, event-driven
    
- Beanstalk = managed app deployment
    

### Hands-On (45 min)

- Launch a basic EC2 (no deep config)
    
- Open Lambda console and inspect sample functions
    

---

## **DAY 4 – Storage & Databases**

**Objective:** Match storage/database to business requirement.

### Study (3 hrs)

#### Storage

- Amazon S3 (classes, durability vs availability)
    
- Amazon EBS
    
- Amazon EFS
    
- Amazon Glacier
    

#### Databases

- Amazon RDS
    
- Amazon DynamoDB
    
- Amazon Aurora
    
- Amazon Redshift
    

### Exam Triggers

- Object storage = S3
    
- NoSQL, serverless DB = DynamoDB
    
- Data warehouse = Redshift
    

### Hands-On (45 min)

- Create an S3 bucket
    
- Upload/download files
    
- Enable versioning
    

---

## **DAY 5 – Security & Identity**

**Objective:** This is **high-weight**—treat seriously.

### Study (3 hrs)

- AWS IAM (Users, Groups, Roles, Policies)
    
- MFA
    
- Shared Responsibility Model
    
- AWS KMS
    
- AWS WAF & Shield
    
- AWS Organizations
    

### Exam Gold Rules

- IAM is **global**
    
- Least privilege always wins
    
- AWS secures the cloud, customer secures data
    

### Hands-On (45 min)

- Create an IAM user
    
- Attach managed policy
    
- Enable MFA
    

---

## **DAY 6 – Pricing, Billing & Support**

**Objective:** Think like a business decision-maker.

### Study (2–3 hrs)

- Pricing models
    
- Free Tier
    
- Savings Plans vs Reserved Instances
    
- Spot Instances
    
- AWS Pricing Calculator
    
- AWS Budgets
    
- Support Plans
    

### Exam Signals

- Long-term workload → Reserved / Savings
    
- Fault-tolerant batch → Spot
    
- Cost visibility → Cost Explorer
    

### Hands-On (30 min)

- Open Billing Dashboard
    
- Explore Cost Explorer & Budgets
