## 🩺 AWS Health Dashboard – CLF-C02 Scope 🌍📊

AWS provides health information through:

- **AWS Service Health Dashboard**
    
- **AWS Personal Health Dashboard**
    

Both are provided by  
Amazon Web Services.

---

# 1️⃣ AWS Service Health Dashboard 🌐

This shows the **overall status of AWS services globally**.

It displays:

- Service outages
    
- Service disruptions
    
- Regional availability
    

👉 Publicly available  
👉 Shows general service status

### Example

If EC2 in Mumbai Region has an outage, it appears here.

---

# 2️⃣ AWS Personal Health Dashboard 🧑‍💻

This shows **issues specific to your AWS account**.

It provides:

- Alerts about resources affected
    
- Scheduled maintenance notifications
    
- Guidance for remediation
    

👉 Account-specific  
👉 More detailed than Service Health Dashboard

---

# 🔥 Service Health vs Personal Health

|Feature|Service Health|Personal Health|
|---|---|---|
|Scope|Global AWS status|Your account only|
|Public?|Yes|No|
|Personalized alerts|No|Yes|

---

# 🎯 Exam Keywords

- “View AWS service outage” → Service Health Dashboard
    
- “See how outage affects my resources” → Personal Health Dashboard
    
- “Account-specific event notification” → Personal Health Dashboard
    

---

## One-Line Memory

> Service Health = Global status  
> Personal Health = Your account status
