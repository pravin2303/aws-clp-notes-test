## Cloud Computing Overview – CLF-C02 Scope ☁️

**Cloud computing** is the on-demand delivery of IT resources over the internet with **pay-as-you-go pricing**.

Instead of buying and maintaining physical servers, you use services from  
Amazon Web Services.

---

## What Cloud Computing Provides

You can access:

- Compute (virtual servers)
    
- Storage
    
- Databases
    
- Networking
    
- Software
    

All without managing physical hardware.

---

## Core Characteristics (Very Important for Exam)

### 1️⃣ On-Demand Self-Service

Launch resources whenever needed.

### 2️⃣ Broad Network Access

Access services over the internet.

### 3️⃣ Resource Pooling

Shared infrastructure among multiple customers.

### 4️⃣ Rapid Elasticity

Scale up or down quickly.

### 5️⃣ Measured Service

Pay only for what you use.

---

## Cloud Service Models

- IaaS (e.g., EC2)
    
- PaaS (e.g., Elastic Beanstalk)
    
- SaaS (Fully managed software)
    

---

## Cloud Deployment Models

- Public Cloud
    
- Private Cloud
    
- Hybrid Cloud
    

---

## Key Benefits

- Lower cost
    
- Scalability
    
- High availability
    
- Global reach
    
- Faster innovation
    

---

## Simple Example

Instead of buying a server:

You launch an EC2 instance in minutes and start running your application.
