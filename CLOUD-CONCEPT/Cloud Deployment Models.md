## Cloud Deployment Models – CLF-C02 Scope ☁️

Cloud deployment models describe **where** and **how** cloud infrastructure is deployed.

For CLF-C02, there are **three main models**.

---

## 1️⃣ Public Cloud 🌍

Infrastructure is owned and operated by a cloud provider like  
Amazon Web Services.

- Resources shared among multiple customers
    
- Pay-as-you-go pricing
    
- Highly scalable
    

**Example:** Running EC2 instances in AWS.

✔ Most common model in AWS exam questions.

---

## 2️⃣ Private Cloud 🏢

Cloud infrastructure used by **only one organization**.

- Can be hosted on-premises
    
- More control
    
- Used for strict compliance
    

✔ More expensive to maintain.

---

## 3️⃣ Hybrid Cloud 🔄

Combination of:

- Public cloud + Private cloud
    

Some workloads run on-premises, others in AWS.

**Example:**  
Sensitive data stored on-premises, website hosted in AWS.

---

# Quick Comparison Table (Exam Friendly)

|Model|Ownership|Access|Example|
|---|---|---|---|
|Public|Cloud Provider|Many customers|AWS|
|Private|Single organization|One company|On-prem data center|
|Hybrid|Both|Mixed|On-prem + AWS|

---

## Exam Keywords

- “Shared infrastructure” → Public cloud
    
- “Single organization only” → Private cloud
    
- “Combination of on-prem and cloud” → Hybrid cloud
    

---

For CLF-C02 remember:

> Public = Shared  
> Private = Single organization  
> Hybrid = Combination
