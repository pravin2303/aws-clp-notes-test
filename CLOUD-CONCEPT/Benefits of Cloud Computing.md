## Benefits of Cloud Computing – CLF-C02 Scope ☁️

Cloud computing (like services from Amazon Web Services) provides several key advantages. These are **core exam concepts** for CLF-C02.

---

## 1️⃣ Trade Capital Expense for Variable Expense 💰

Instead of buying physical servers (CapEx):

- You pay only for what you use (OpEx)
    
- No upfront hardware cost
    

👉 Example: Launch EC2 instead of buying a server.

---

## 2️⃣ Benefit from Massive Economies of Scale 📉

Cloud providers operate at huge scale.

- Lower cost per resource
    
- Savings passed to customers
    

---

## 3️⃣ Stop Guessing Capacity 🤔

No need to predict future traffic.

- Scale up when traffic increases
    
- Scale down when traffic decreases
    

👉 Uses services like Auto Scaling.

---

## 4️⃣ Increase Speed and Agility ⚡

Resources can be provisioned in minutes.

- Launch servers instantly
    
- Faster innovation
    
- Faster development
    

---

## 5️⃣ Stop Spending Money Running Data Centers 🏢

No need to manage:

- Power
    
- Cooling
    
- Hardware maintenance
    

Cloud provider handles infrastructure.

---

## 6️⃣ Go Global in Minutes 🌍

Deploy applications worldwide using:

- Multiple Regions
    
- Availability Zones
    
- Edge locations
    

---

# Quick Exam Summary

AWS defines **6 main benefits**:

1. Trade CapEx for OpEx
    
2. Economies of scale
    
3. Stop guessing capacity
    
4. Increase speed & agility
    
5. Stop running data centers
    
6. Go global quickly
    

---

For CLF-C02 remember:

> Cloud = Cost savings + Scalability + Agility + Global reach
